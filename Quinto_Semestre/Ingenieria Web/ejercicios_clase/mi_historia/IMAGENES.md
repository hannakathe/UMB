## Notas de generación de imágenes

### Herramientas recomendadas:
- **Midjourney v6**: Usar los prompts tal cual, añadir `--ar 16:9` para landscape o `--ar 5:6` para portrait
- **DALL-E 3 (ChatGPT)**: Los prompts funcionan bien directamente
- **Stable Diffusion XL**: Añadir al final: `masterpiece, best quality, detailed`
- **Ideogram**: Muy bueno para imágenes con texto en pantalla (Imágenes 2, 3, 6, 7)

### Recomendaciones de estilo globales:
- Pedir siempre: `dark background`, `cyberpunk`, `sci-fi`
- Evitar: `bright colors`, `cheerful`, `cartoon style`
- Añadir si es posible: `cinematic lighting`, `atmospheric`, `moody`
