## Notas de generación

### Herramientas recomendadas:
- **Midjourney v6**: Usar los prompts tal cual, añadir `--ar 16:9` para landscape o `--ar 5:6` para portrait
- **DALL-E 3 (ChatGPT)**: Los prompts funcionan bien directamente
- **Stable Diffusion XL**: Añadir al final: `masterpiece, best quality, detailed`
- **Ideogram**: Muy bueno para imágenes con texto en pantalla (imágenes 2, 3, 5, 6)

### Recomendaciones de estilo globales:
- Pedir siempre: `dark background`, `cyberpunk`, `sci-fi`
- Evitar: `bright colors`, `cheerful`, `cartoon style`
- Añadir si es posible: `cinematic lighting`, `atmospheric`, `moody`

### Archivos de media (audio/video) también necesarios:
| Archivo | Descripción | Dónde obtener |
|---|---|---|
| `alarm.mp3` | Alarma de sistema / alerta de emergencia | freesound.org, search "system alert" |
| `typing.mp3` | Sonido de teclado mecánico escribiendo | freesound.org, search "keyboard typing" |
| `nexus-ambient.mp4` | Video de interfaz futurista / data visualization | Pexels, search "data visualization loop" o "HUD interface" |
